# Wesnoski Foundations · Resource Library

Free educational resource published at https://resource.wesnoskicoaching.com

Built and maintained by Coach Farren Brown of Wesnoski Coaching.

## Contents

29 docs across 6 sections covering the MARCH method, energy, weight, gut, hormones, and anonymized case studies. Includes three quizzes: phase diagnostic, knowledge check, and coaching fit check.

## Deploy

This site is deployed via GitHub Pages with a custom domain. Single-file HTML, no build step required.

To update content, edit `index.html` and commit. GitHub Pages republishes within 1 to 2 minutes.
